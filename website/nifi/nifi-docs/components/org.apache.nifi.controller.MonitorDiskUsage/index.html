Description:

Checks the amount of storage space available for the Content Repository and FlowFile Repository and warns (via a log message and a System-Level Bulletin) if the partition on which either repository exceeds some configurable threshold of storage space

Tags:

disk, storage, warning, monitoring, repo

Properties:

In the list below, the names of required properties appear in bold. Any other properties (not in bold) are considered optional. The table also indicates any default values.

NameDefault ValueAllowable ValuesDescription
Content Repository Threshold80%The threshold at which a bulletin will be generated to indicate that the disk usage of the Content Repository is of concern
FlowFile Repository Threshold80%The threshold at which a bulletin will be generated to indicate that the disk usage of the FlowFile Repository is of concern
